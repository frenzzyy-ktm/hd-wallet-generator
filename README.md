# HD Wallet Address Generator (BIP44)

HD Wallet Address Generator is a Node.js-based script for generating **Hierarchical Deterministic (HD)** wallet addresses from a mnemonic (seed phrase) using the **BIP44** standard.

This project is intended for **educational, testing, and development purposes** only and **does not store any user data**.
